
## Reference

### Extending

To customize textarea styles, extend the component using `@layer components`:

```css
@layer components {
    textarea,
    [is-~="textarea"] {
        &[size-="thicc"] {
            min-height: 6lh;
            padding: 2lh 4ch;
        }

        /* Add more custom styles here */
    }
}
```

### Scope

* All native `<textarea>` elements
* Any element with `is-~="textarea"`

```css
textarea,
[is-~="textarea"] {
  /* ... */
}
```
